runApp :: IO () Source #

Run the app by loading items from the config file, turning them into an initial state and then delegating to Brick. Unless a yaml parsing exception is printed to facilitate config file debugging, there is no failure condition and the program can exit successfully. This situation may change after permission issues are considered.
